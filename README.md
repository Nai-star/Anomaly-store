# SAYRAY: TURNO DE NOCHE

**Convenience Store · Horror Survival · 3D Browser Game**

![Three.js](https://img.shields.io/badge/Three.js-000000?style=flat&logo=three.js&logoColor=white)
![React](https://img.shields.io/badge/React-19-61DAFB?style=flat&logo=react&logoColor=white)
![Vite](https://img.shields.io/badge/Vite-8-646CFF?style=flat&logo=vite&logoColor=white)

Sayray: Turno de Noche es un juego de terror en primera persona donde trabajás el turno de madrugada en una tienda de conveniencia. Mantené la tienda operativa mientras fenómenos anómalos ocurren a tu alrededor.

### 🎮 Jugabilidad
- **Turno**: 12:00 AM - 6:00 AM (tiempo real comprimido)
- **Tareas**: Encender luces, contestar teléfono, limpiar baño, operar caja
- **Anomalías**: Maniquí flotante, luces rojas, objetos movidos, cajas extrañas
- **Controles**: WASD (moverse), Mouse (mirar), E (interactuar)

### 🚀 Inicio rápido
```bash
cd frontend
npm install
npm run dev
```
Abrir en `http://localhost:5173`

### 🛠 Stack
- **Frontend**: React 19, Three.js, @react-three/fiber, Rapier physics
- **Build**: Vite 8
- **Backend (opcional)**: Django 6 / Express 5
